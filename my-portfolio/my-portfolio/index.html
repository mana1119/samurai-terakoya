WORKS

SKILLS

HTML/CSS

HTMLでは、意味のあるタグを使ってサイトの内容がわかりやすいように組み立てます。
CSSでは、見た目を整えつつ、スマホでもきれいに見えるように工夫しています。

JavaScript

動きのあるインタラクティブなページを作成できます。
フォームの入力チェックやスライダー、メニューの開閉など、ユーザー体験を向上させる機能を実装します。

WordPress

カスタマイズ可能なサイトやブログを構築できます。
テーマ編集やプラグイン導入で、使いやすく魅力的なサイト作りをサポートします。

Figma

画面設計やプロトタイプを効率的に作成できます。
チームとリアルタイムで共有・編集できるため、スムーズなデザイン制作が可能です。

Photoshop

高品質な画像編集やレタッチが可能です。
バナー制作やWeb素材の作成など、デザインの細部までこだわった制作を行います。

PROFILE

Webサイトのデザインからコーディング、運用まで一貫して対応。
成果につながる信頼感のあるサイトを、シンプルで洗練されたデザインで丁寧に制作します。
